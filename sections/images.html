Utiliser la balise <img />

La balise <img /> est utilisée pour afficher une image dans une page HTML. Elle est auto-fermante et nécessite au minimum un attribut src pour spécifier l'URL de l'image.

Exemple :

<img src="https://pixabay.com/fr/images/download/raccoon-8504925_640.png" />

Attributs optionnels

En plus de src, d'autres attributs peuvent être utilisés avec la balise <img /> :

  • alt : Donne une description alternative de l'image pour les utilisateurs malvoyants ou lorsque l'image ne peut pas être chargée.
  • width : spécifie la largeur de l’image en pixels. La hauteur s’ajuste en fonction.

Exemple :

<img src="https://pixabay.com/fr/images/download/raccoon-8504925_640.png" 
     alt="Un raton laveur mignon sort la tête d'une poubelle les yeux écarquillés"
     width="300" />
