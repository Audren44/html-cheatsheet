Ajouter un bloc de code en HTML

On peut indiquer qu'un bloc contient du code dans un langage de programmation en utilisant les balises <pre> et <code>. La balise <code> est utilisée pour le code en ligne, tandis que la balise <pre> conserve la mise en forme (espaces, sauts de ligne, etc.) d'un bloc de code entier.

Attention : Si on essaie d'écrire du code HTML dans un document HTML, le navigateur va l'interpréter comme du code HTML.

Pour éviter cela, on remplace :

  • & par &amp;
  • < par &lt;
  • > par &gt;
  • " par &quot;

Par exemple, pour obtenir ce morceau ci-dessous :

	
		<pre>
			<code>
				<h1>Hello, world</h1>
				<h2>Welcome to my Website</h2>
			</code>
		</pre>

J'ai dû écrire dans mon éditeur :

	
		&lt;pre&gt;
			&lt;code&gt;
				&lt;h1&gt;Hello, world&lt;/h1&gt;
				&lt;h2&gt;Welcome to my Website&lt;/h2&gt;
			&lt;/code&gt;
		&lt;/pre&gt;

Bloc de citation : <blockquote>

La balise <blockquote> est utilisée pour représenter une citation longue provenant d'une autre source. Elle est généralement stylisée différemment pour indiquer qu'il s'agit d'un extrait de texte cité.

Exemple de balise <blockquote> :

				<blockquote cite="https://fr.wikipedia.org/wiki/Chat">
					Les chats sont les membres de la famille des félins, 
					et ils partagent de nombreuses caractéristiques avec 
					leurs cousins les lions, les tigres et les panthères.
				</blockquote>

Attribut cite

L'attribut cite permet de spécifier l'URL de la source d'origine de la citation, offrant ainsi un lien vers le texte original.
