Les Paragraphes

En HTML, les paragraphes sont créés à l'aide de la balise <p>. Ils permettent de structurer le texte en blocs lisibles. En ce moment-même tu es en train de lire un paragraphe. Avec les balises ça donne : 

                    <p>Exemple de paragraphe</p>

Ajouter un commentaire

Les commentaires en HTML sont ajoutés avec la syntaxe suivante :

				<!-- Je suis un commentaire -->

Les commentaires ne sont pas affichés dans le navigateur, mais ils aident à documenter le code.

Donner de l’importance à une partie du texte

La balise <strong> est utilisée pour indiquer qu'un texte est important. Par défaut, cela rend le texte en gras. 

                    <strong>Texte important</strong>

Voici un exemple :

Il est important de bien structurer votre code HTML.

Inline vs Block

Il y a deux catégories d'éléments en HTML : les éléments dits "inline" (en ligne) et les éléments de type "block". Les éléments inline s’affichent sur la même ligne (ex : <strong>). Les élements block prennent toute la largeur de la page (ex : <h1>)

Emphase

Le texte dans la balise <em> s'affiche en italique et est lu comme un élément remarquable.

Exemple : les chats sont les membres de la famille des félins, et ils partagent de nombreuses caractéristiques avec leurs cousins les lions, les tigres et les panthères.

Les balises <b>, <i>, <u> - Gras, italique, souligné

Les balises <b> (gras), <i> (italique), et <u> (souligné) sont souvent utilisées pour modifier le style des éléments textuels. Cependant, elles ne fournissent aucune signification sémantique et peuvent poser des problèmes d'accessibilité et de référencement.

Pourquoi éviter ces balises ?

  • Ces balises n'indiquent pas le but ou la fonction du contenu, ce qui peut rendre difficile la compréhension pour les lecteurs d'écran.
  • Pour le référencement, les moteurs de recherche ne peuvent pas interpréter ces balises de manière significative, ce qui peut affecter le SEO.

Alternatives sémantiques

Il est préférable d'utiliser des balises sémantiques pour donner un sens au contenu :

  • <strong> : pour indiquer une importance accrue (équivalent au gras).
  • <em> : pour indiquer une emphase (équivalent à l'italique).
  • <mark> : pour mettre en évidence un texte.

Balises d'accessibilité

Certaines balises HTML sont conçues pour améliorer l'accessibilité de vos pages et fournir des informations supplémentaires aux utilisateurs et moteurs de recherche.

<abbr> - Abréviation

La balise <abbr> est utilisée pour indiquer qu’un mot ou une expression est une abréviation. Elle améliore l'accessibilité et le référencement, car elle permet de voir le texte complet de l'abréviation en survolant avec la souris. 

				<p>
					Site réalisé en <abbr title="HyperText Markup Language">HTML</abbr>,
					<abbr title="Cascading Style Sheets">CSS</abbr> et
					<abbr title="JavaScript">JS</abbr>.
				</p>

Exemple d'affichage : Site réalisé en HTML, CSS et JS.

Ajouter un bloc de code en HTML

On peut indiquer qu'un bloc contient du code dans un langage de programmation en utilisant les balises <pre> et <code>. La balise <code> est utilisée pour le code en ligne, tandis que la balise <pre> conserve la mise en forme (espaces, sauts de ligne, etc.) d'un bloc de code entier.

Attention : Si on essaie d'écrire du code HTML dans un document HTML, le navigateur va l'interpréter comme du code HTML.

Pour éviter cela, on remplace :

  • & par &amp;
  • < par &lt;
  • > par &gt;
  • " par &quot;

Par exemple, pour obtenir ce morceau ci-dessous :

	
		<pre>
			<code>
				<h1>Hello, world</h1>
				<h2>Welcome to my Website</h2>
			</code>
		</pre>

J'ai dû écrire dans mon éditeur :

	
		&lt;pre&gt;
			&lt;code&gt;
				&lt;h1&gt;Hello, world&lt;/h1&gt;
				&lt;h2&gt;Welcome to my Website&lt;/h2&gt;
			&lt;/code&gt;
		&lt;/pre&gt;

Bloc de citation : <blockquote>

La balise <blockquote> est utilisée pour représenter une citation longue provenant d'une autre source. Elle est généralement stylisée différemment pour indiquer qu'il s'agit d'un extrait de texte cité.

Exemple de balise <blockquote> :

				<blockquote cite="https://fr.wikipedia.org/wiki/Chat">
					Les chats sont les membres de la famille des félins, 
					et ils partagent de nombreuses caractéristiques avec 
					leurs cousins les lions, les tigres et les panthères.
				</blockquote>

Attribut cite

L'attribut cite permet de spécifier l'URL de la source d'origine de la citation, offrant ainsi un lien vers le texte original.
