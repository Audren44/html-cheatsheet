Ajouter des vidéos

  • <video> : balise utilisée pour ajouter un fichier vidéo sur une page web. Pour des raisons d'accessibilité, il faut toujours laisser la possibilité aux utilisateurs de démarrer/arrêter la vidéo et d'augmenter ou baisser le volume. On peut spécifier un texte à l'intérieur de la balise qui sera visible si le navigateur ne supporte pas l'élément <vidéo>. On peut trouver pleins de vidéos gratuites et libres de droits sur le site de pexels.
  • src : attribut pour spécifier la source de la vidéo.
  • controls : attribut (booléen) pour afficher (ou pas) le panneau de contrôle du lecteur vidéo.
  • autoplay : attribut (booléen) jouera automatiquement la vidéo (déconseillé pour ne pas déranger les utilisateurs).
  • loop : attribut (booléen) qui permet de lancer la vidéo en boucle : lorsque la vidéo est terminée, la lecture reprendra depuis le début.
  • muted : attribut (booléen) qui permet de couper le son par défaut.
  • poster : lien vers une image
    • qui permet de définir une image affichée avant la lecture de la vidéo.
  • width et height : nombre de pixels permettent de définir la taille de la vidéo.

Exemple d'une balise video avec des attributs optionnels :

			
	<video src="cats.mp4" controls autoplay loop muted poster="cats_video_poster.png" width="300">
	   <p>Impossible de jouer la vidéo. Vous pouvez <a href="cats.mp4">télécharger la vidéo</a> afin de la visionner.
	   </p>
	</video>
  • <track> : balise pour ajouter du texte dans les vidéos.
  • kind="subtitles" : attribut pour ajouter des sous-titres.
  • kind="captions" : attribut pour ajouter des légendes.
  • kind="descriptions" : attribut pour ajouter des descriptions.
  • default : attribut qui force l’affichage des sous-titres dès que la vidéo est lue (certains navigateurs n’affichent pas les sous-titres sans l'attribut default.)

Exemple d'une balise <track> inserée dans une balise <video> : 

			
	<video src="cats.mp4" controls autoplay loop muted poster="cats_video_poster.png" width="300">
	<track kind="captions" src="/assets/subtitles/my_captions.vtt" srclang="fr" default />
	   <p>Impossible de jouer la vidéo. Vous pouvez <a href="cats.mp4">télécharger la vidéo</a> afin de la visionner.
	   </p>
	</video>
  • fichier WebVTT : le texte dans les vidéos est ajouté à l'aide d'un fichier WebVTT : c'est un fichier avec l'extension .vtt qui contient les textes associés à un minutage. Il existe des programmes qui permettent de créer facilement des fichier VTT, comme vtt-creator par exemple. Ce fichier doit avoir une structure précise.

Exemple d'un fichier .vtt :

			
WEBVTT

00:00:00.000 --> 00:00:03.000
Je suis le roi de la jungle

00:00:03.000 --> 00:00:06.000
Je suis un killer
