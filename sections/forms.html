Les Boutons

Les boutons sont créés avec la balise <button>. L'attribut disabled peut être utilisé pour désactiver un bouton.

On peut utiliser l'attribut avec une valeur booléenne :

                <button disabled="true">Clique</button>

Cependant, il est possible de raccourcir en utilisant l'attribut sans aucune valeur : 

                <button disabled>Clique</button>
